Wrapped agent.stream() with exponential backoff. On a rate-limit it now retries up to 3× before surfacing the error.

async function withRetry(run) {
  for (let i = 0; i < 3; i++) {
    try { return await run(); }
    catch (e) { if (e.status !== 429) throw e; }
    await sleep(2 ** i * 500);
  }
}
How can I help you today?

Why Relay

A frontier agent, without the lock-in.

⌘

Bring your own model

Wired to Mastra's model router — every provider, hundreds of models. Pick OpenAI, Anthropic, DeepSeek, OpenRouter, or local. Search and switch in one keystroke.

⛬

Your keys, encrypted

API keys are stored with the OS keychain via Electron safeStorage — encrypted on disk, never in a config file, never sent anywhere but the provider you chose.

◇

Truly open

MIT-spirited, no telemetry, no account. The whole thing is a few thousand lines of TypeScript you can read in an afternoon and bend to your workflow.

Two modes, one brain

Talk it through, or hand it the keyboard.

Chat

Think out loud

A focused, streaming conversation for the moments before you write code — design decisions, gnarly bugs, the “why is this slow” rabbit holes. Sessions are saved automatically, the instant you send your first message.

Code

Ship the change

Point the same agent at your repository and let it read, grep, and edit. The pastel timeline shows every step it takes so nothing happens behind your back.

Any provider

One picker. Every model worth running.

Relay reads the provider registry locally — no network call to list models — so the picker is instant even with hundreds of options.
